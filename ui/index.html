Серия

Части одной большой истории с общим контекстом. Номер части назначается автоматически при создании из списка серии.

Задание для генерации переписки через ChatGPT (OpenRouter). Например: «Часть 3: Даня палится современными словами, Алиса учит советскому языку».

Когда диалог уже есть, опишите что изменить: тон, финал, конкретные реплики. Текущий JSON уйдёт в модель как основа.

Тема мессенджера

Для «Сюжет в кадре» тема не влияет на видео — пузыри рисуются поверх кадра в фиксированном стиле.

Текст в пузырях и строка ввода (px в кадре 1080×1920). Пусто — по умолчанию 53.

Подача текста

Переписка — мессенджер в горизонтальном кадре. Повествование — крупный текст на экране, по одной реплике.

Формат видео
Анимация сюжета

Depth parallax — сдвиг по карте глубины (цикл 3 с). Ken Burns — лёгкий наезд без depth.

Переход между кадрами

На стыке story-сцен: zoom — наезд и вспышка, push — сдвиг, crossfade — мягкое растворение.

Цветовой фильтр

Накладывается на story-кадры и Veo-анимацию: тёплый, холодный, кино, винтаж и др.

Заставки

Выберите трек из каталога или загрузите свой MP3. При рендере на GPU-воркер трек отправляется вместе с озвучкой.

Одно значение на весь диалог: паузы, набор и задержки после сообщений. Меньше — быстрее, больше — медленнее.
