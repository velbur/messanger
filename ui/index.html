Chat Video Generator

Редактор переписок, сохранение в локальную базу и сборка MP4 для Shorts

Сериалы

Каждый сериал — отдельный контекст. Откройте серию, чтобы выбрать часть.

Части одной истории. При генерации учитывается контекст только предыдущих частей этой серии.

Shorts

Отдельные переписки без связи между собой.

Для сериала — латиница, например poka_v_sssr_part3

В базе и в файлах рендера: json/<имя>.json, out/<имя>.mp4

Серия

Части одной большой истории с общим контекстом. Номер части назначается автоматически при создании из списка серии.

Задание для генерации переписки через ChatGPT (OpenRouter). Например: «Часть 3: Даня палится современными словами, Алиса учит советскому языку».

Когда диалог уже есть, опишите что изменить: тон, финал, конкретные реплики. Текущий JSON уйдёт в модель как основа.

Изображения

Кнопка активна, если в JSON есть фото-сообщения с imagePrompt, но без прикреплённого файла image. Промпт каждой сцены строится по контексту всей переписки.
