Chat Video Generator

Редактор переписок, сохранение в локальную базу и сборка MP4 для Shorts

В базе и в файлах рендера: json/<имя>.json, out/<имя>.mp4

Сообщения как в видео. Там, где в JSON есть image, — блок фото: Kling или Grok у каждого кадра, прикрепить, сгенерировать, промпт. Клик по превью — полный размер.

Сохранённые диалоги

Локальная база data/dialogues.db. Откройте диалог в редакторе, измените, снова сохраните или соберите видео.

Лимиты и настройка API

Остатки пакетов Kling запрашиваются с официального API. У xAI (Grok) в этом проекте нет эндпоинта баланса — только статус ключей и модели.

Общий стиль для всех кадров (генерация и «Промпт от Grok»). После смены стиля у уже заполненных кадров нажмите Промпт от Grok заново или очистите «Промпт для этого кадра».
